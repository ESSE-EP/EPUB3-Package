EPUB3 Package
=============

EPUB3 package for ESSE-EP tutorials as well as for those who just need a quick basic EPUB3 package.
This package contains the standard directories and files.
  - EPUB
    - package.opf
    - XHTML
      - nav.xhtml = Table of Contents
    - CSS
      - styles.css
    - JS
      - script.js
  - META-INF
    -container.xml
  - mimetype
  
Develop, have fun!
